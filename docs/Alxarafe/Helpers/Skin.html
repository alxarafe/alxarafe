class Skin

Class Skin

Constants

SKINS_FOLDER

Default: It is the folder that includes the templates.

Each template will be a folder whose name will be the one that will appear in the template selector.
COMMON_FOLDER

Default: Folder of the common code.

It is where the common files will be placed to all the templates. If a file is not found in the template, then it will be searched in COMMON_FOLDER.
